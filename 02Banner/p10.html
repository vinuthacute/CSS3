CSS-3 RADIAL-GRADIENTS..!

In a radial gradient color image emerge from a single point and smoothly spread outward ina circular or elliptical shape rather than fading from one color to another in a single direction as with linear gradients. The basic syntax of creating a radial gradient can be with

SYNTAX : radial-gradient(Shape size at position, color-stop1, color-stop2,.....);

The arguments of the radial-gradient() function has the following meaning:

  1. position - Specifies the starting point of the gradient, which can be specified in units (px, em, or percentages) or keyword(left, bottom, ect).
  2. shape - Specifies the shape of the gradient's ending shape. It can be circle or ellipse.
  3. size - Specifies the size of the gradient's ending shape. It can be The default is farthest.
